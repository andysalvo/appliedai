# Applied AI Club at Penn State

The Applied AI Club is a student organization at Penn State dedicated to helping students understand how AI is changing the way businesses operate. We run guest speaker sessions, hands-on workshops, and case competitions. For members who want to go deeper, our Labs program takes on full application builds using industry-standard development workflows.

**Website:** [appliedaipennstate.com](https://appliedaipennstate.com)

## Contributing

This repository powers the club's website and workspace. Members contribute through a conversational AI workspace inside GitHub Codespaces. No coding experience required.

1. Open this repository on GitHub
2. Click **Code** > **Codespaces** > **Create codespace on main**
3. The workspace opens automatically. Talk to the agent to contribute.

For technical contributors, see [AGENTS.md](AGENTS.md) for code style, branching, and CI/CD details.

## Programs

- **Guest Speakers and Workshops** — Professionals from all fields share how AI has changed the way they work. Hands-on workshops and case competitions.
- **Labs** — Full application builds using the same tools and workflows used in professional engineering teams.
- **Tool Registry** — A list of AI tools worth knowing, organized for students at every level.
